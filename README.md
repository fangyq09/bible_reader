一个轻量级的圣经阅读器。

专注于阅读圣经。

直接下载二进制程序，将其放在 /usr/local/bin/下面或~/.local/bin/下。
圣经数据库放置到~/.local/share/bible_reader/sqlite/文件夹中即可。

用户笔记自动存储于~/.local/share/bible_reader/notes/note.db数据库中。
