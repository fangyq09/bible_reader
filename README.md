# 一个轻量级的跨平台圣经阅读器。


- 适合人群：专注于阅读圣经，以及有少量笔记需求者。


- Linux上的安装： 直接下载Linux平台的二进制程序，将其解压放在 /usr/local/bin/下面或~/.local/bin/下， 加上可执行权限即可。

  例如 sudo cp br /usr/local/bin/ 

  然后 sudo chmod +x /usr/local/bin/br

  在终端输入软件名 br 回车即可启动程序。

- Window上的安装：直接下载Windows版的程序，解压出来，双击即可运行（免安装）。


- 如果需要添加更多圣经数据库， 请将数据库放置到~/.local/share/bible_reader/sqlite/文件夹中即可。


- 用户笔记自动存储于~/.local/share/bible_reader/notes/note.db数据库中，如果需要备份笔记数据，只需备份note.db。


- 快捷键 Ctrl i / Ctrl o 可以在浏览过的历史界面之间前后转跳。
- 搜索经文：可以直接输入关键词“XXX”进行整本圣经搜索，也可以“书卷名：XXX”这种只搜索某卷书
